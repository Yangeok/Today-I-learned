# 스위치란

- L1 (허브, 더미허브)
  - 랜선에 꽂힌 모든 포트에 데이터를 전부 다 전달한다
- L2 (스위칭허브)
  - 패킷의 mac주소를 읽어 스위칭한다
  - 구조가 간단하며 신뢰성이 높다
  - 패킷에 의해 성능 저하가 발생한다
- L3 (라우터, 공유기)
  - L2 스위치에 라우팅 기능이 추가됨
  - 스위치 자체에 IP 주소가 할당된다
- L4
  - 외부에서 들어오는 모든 요청은 L4 스위치를 거친다
  - 주요 역할은 로드밸런싱이다
  - 주로 A/A 혹은 A/S 기능을 사용한다
- L7
  - url, 쿠키 등을 분석해 보안에 유리하게 정교한 로드밸런싱이 가능하다
